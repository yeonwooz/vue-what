# vue3-project

## 프로젝트 시작(웹서버, db서버 실행)

```
npm start

// 웹서버, db서버 각각 실행하고 싶은 경우
npm run serve  // 웹서버 실행
npm run watch  // db서버 실행
```

### Lints and fixes files

```
npm run lint
```
